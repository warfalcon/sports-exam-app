

# 体育中考应用 (Sports Exam App)

一款专为体育中考设计的移动端 Web 应用，帮助学生记录日常体育锻炼数据、查看成绩标准和跟踪训练进度。

## 功能特性

### 📊 数据记录
- 支持多种运动项目成绩录入（800米/1000米、50米跑、仰卧起坐/引体向上、立定跳远、坐位体前屈）
- 每日运动记录保存与历史查询
- 一键快速签到功能

### 📈 数据可视化
- 训练趋势图表展示
- 连续打卡天数统计
- 成绩变化历史曲线

### 🎯 成绩评估
- 内置各项目评分标准（男/女生分开）
- 根据输入成绩自动计算得分
- 成绩对比和模拟预测

### 🔐 用户系统
- 用户注册与登录
- 数据本地存储，保护隐私

### ⚙️ 个性化设置
- 主题颜色切换
- 考试日期倒计时
- 数据导出功能

## 技术栈

- **前端框架**: 原生 JavaScript（无框架依赖）
- **存储**: LocalStorage 本地存储
- **图表**: Chart.js
- **PWA**: Service Worker + Manifest 支持离线使用

## 快速开始

### 方法一：直接使用
在浏览器中打开 `index.html` 文件即可使用。

### 方法二：本地服务
```bash
# 使用 Python 启动简易服务器
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

，然后在浏览器访问 `http://localhost:8080`

### 方法三：部署到服务器
将所有文件上传到任意 Web 服务器即可访问。

## 评分标准

应用内置了完整的体育中考评分标准，包括：

### 女生项目
- 800米跑
- 50米跑
- 仰卧起坐
- 立定跳远
- 坐位体前屈

### 男生项目
- 1000米跑
- 50米跑
- 引体向上
- 立定跳远
- 坐位体前屈
